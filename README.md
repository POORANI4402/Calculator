# 🧮 Calculator Web App (Light/Dark Mode)
# 📌 Project Overview

This project is a fully functional and interactive calculator web application developed using HTML5, CSS3, and JavaScript. It is designed to perform real-time arithmetic calculations while providing an enhanced user experience through a clean interface, responsive design, and dynamic theme switching.

The application simulates the behavior of a standard calculator and includes additional usability features such as keyboard input support, live screen updates, and visual feedback for user actions.

# 🚀 Key Features

🔢 Core Functionalities

Perform basic arithmetic operations: addition (+), subtraction (−), multiplication (×), and division (÷) 

Real-time input display as users type or click buttons

Supports decimal calculations

Handles invalid operations like division by zero with proper error messages

🌗 Theme Switching (Light/Dark Mode)

Users can toggle between light and dark themes

Dynamically switches between light.css and dark.css

Theme icon changes (☀️ Sun / 🌙 Moon) based on active mode

Smooth visual transition enhances user experience

⌨️ Keyboard Support

Users can input values using keyboard keys (0–9, +, −, *, /, .)

“Enter” key evaluates the result

“Backspace” removes the last entered value

Prevents default browser behavior for better control

🧠 Smart Input Handling

Continuous input concatenation using JavaScript

Backspace functionality implemented using string manipulation

Error handling using isNaN() for invalid expressions

📱 Responsive UI Design

Built using CSS3 Flexbox/Grid

Optimized for desktop and mobile devices

Clean and modern layout for better usability

⚙️ Technical Implementation

🧩 HTML5

Structured layout including calculator buttons, display screen, and theme toggle

Semantic elements for better readability and maintainability

🎨 CSS3

Separate stylesheets for light and dark themes

Use of transitions for smooth theme switching

Responsive design techniques for different screen sizes

⚡ JavaScript

Core logic handled in script.js

calculate() function evaluates expressions using eval()

changeTheme() dynamically updates styles and icons

liveScreen() updates display instantly

keyboardInputHandler() manages all keyboard interactions
